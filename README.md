# TAPATALK - Guia de Instalacao e Uso

Este pacote contem:
- `TAPATALK-Setup.exe` (instalador)

## Requisitos
- Windows 10 ou Windows 11 (64-bit)

## Como instalar
1. Baixe `TAPATALK-Setup.exe`.
2. Execute o instalador.
3. Siga o assistente.
4. (Opcional) Marque:
   - criar atalho na area de trabalho
   - iniciar o TAPATALK com o Windows
5. Finalize e abra o TAPATALK.

## Como usar
1. Abra o Microsoft Teams.
2. Confirme o icone do TAPATALK na bandeja do sistema (perto do relogio).
3. Clique com o botao direito no icone para abrir o menu.
4. Opcoes principais:
   - `Protecao ativa`: liga/desliga imediatamente.
   - `Apenas quando apresentar`: mascara apenas durante compartilhamento.
   - `Modo`:
     - `Cobertura solida` (mais seguro)
     - `Pixelar janela`
     - `Imagem`
   - `Desativar protecao por 5 min`: pausa temporaria.
   - `Sobre`: mostra autor e links.

## Recomendacao de seguranca
- Para reunioes com dados sensiveis, prefira `Modo -> Cobertura solida`.
- Teste rapidamente antes da reuniao para validar seu cenario de monitor/tela.

## Atualizar para nova versao
1. Feche o TAPATALK na bandeja (`Sair`).
2. Execute o novo `TAPATALK-Setup.exe`.
3. Conclua o instalador.

## Desinstalar
- Windows > Configuracoes > Aplicativos > TAPATALK > Desinstalar

## Solucao rapida de problemas
- Mascara ficou presa:
  1. Clique no icone da bandeja.
  2. Desmarque `Protecao ativa`.
  3. Clique `Sair` e abra o TAPATALK novamente.
- `Apenas quando apresentar` nao acionou:
  - Desative e ative novamente a opcao no menu.
  - Use temporariamente `Protecao ativa` manual.
